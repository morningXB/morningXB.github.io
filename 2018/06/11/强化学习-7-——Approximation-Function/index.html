强化学习(7)——Approximation Function

发表于 2018-06-11 | 分类于 强化学习 | 本文总阅读量 次

Introduction

1.why we need Approximation Function？
当状态空间过大，或者状态空间连续时，难以用Q表格来解决问题，所以我们需要一个函数称为近似函数，传入当前状态以及动作计算得到Q的近似值，通过有限的状态动作对，我们拟合参数，从而计算未出现的状态动作对Q值。
2.what is the type of Approximation Function？
近似函数有线性和非线性之分，常用的类型有用线性回归拟合的线性函数和用反向传播拟合的人工神经网络。
3.the common learning method？
增量法即随机梯度下降法，注重在线学习，其次是批处理法即批梯度下降法，注重的是数据利用的高效性。
4.the converge of Approximation Function？
对于MC来说，线性非线性函数的预测都可以收敛，而TD在非线性上难以收敛，原因是MC总是提供奖励回馈的无偏估计，而TD采用带偏差的估计值。对于Control问题，他们在非线性函数都难以收敛，其他具体表现下面会列出。为了解决非线性函数不收敛的问题deepmind提出了DQN(Deep Q-learning Network)方法，将深度学习与强化学习结合，在之后的章节会专门讲讲DQN。

批处理法

增量法即随机梯度下降，但是每次一个样本的训练并且不会重复使用，导致数据不能高效利用，所以采用批梯度下降，即最小二乘法优化。一次采集一批数据，并且采用批梯度下降公式即可。
也有一种方法依然是对于一批数据，循环采用随机梯度下降，最终也会逼近最小二乘的优化参数值。
对于线性近似函数甚至可以令更新值等于0，直接计算出参数w。
